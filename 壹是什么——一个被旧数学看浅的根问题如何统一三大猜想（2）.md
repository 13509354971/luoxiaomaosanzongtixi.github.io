## 摘要

现有数学建立在“孤立一”的认知接口之上，而非本源数学。本文将“壹的本体结构”作为唯一公理起点：本源壹由两种同质反向之态归一而成。以此单点修正后，数的根本分类需重写：2 为并置位、3 为唯一骨架数、6 为最小闭锁周期；进而对“走廊”“π = 3 + δπ”“e = 2 + δe”“黎曼临界线 1/2”“哥德巴赫配对”的同根性给出结构性解释与接口对齐草图。本文作为切口稿，聚焦最短链路与可证伪边界；完整推导链与证明骨架见原著[10]。

## 一、根问题只有一个：壹是什么（公理层）

### 1.1 唯一公理：本源壹

Axiom 本源壹是本体系的唯一公理实体，其经典数值接口记为 1。

### 1.2 双态归一：壹的本体结构

Def（双态归一） 本源壹由两种同质反向之态构成：内态与外态。记态半符为 ½⃝，归一算子为 ⊕。定义本体归一：

1 ≙ ½⃝ ⊕ ½⃝

其中 ½⃝ 非数值 1/2；⊕ 非外部加法；≙ 为结构归一接口。为避免混淆，本文用“=”仅标数值等号，“≡”作定义/接口同一，“≙”作结构归一。

### 1.3 物理对应（说明，非证明）

量子纠缠可理解为双态归一的现象域投影；本文不以此作数学证明，仅提示同源性。

## 二、本源确定后，数如何重分（定义层）

接口对齐约定（冻结） 每一结构概念给出（a）结构定义，（b）经典接口；当二者一一等价时，以“接口对齐”标注检核式。

### 2.1 起源数 0：序位原点

Def 0 ≡ Origin。0 为数轴定位原点，不参与正生成域内的承接判定。

### 2.2 本源壹 1：唯一公理实体

Def 1 ≡ 𝟏（本体）且 1 ≙ ½⃝ ⊕ ½⃝。1 不为质数、不为合数，不进入承接。

### 2.3 并置位 2：双态并置

Def 2 ≡ 1 + 1（经典接口）。结构解释：两个本源壹并置，各自保持双态完整，形成最小稳定并置结构。2 是并置位（非骨架）。

### 2.4 唯一骨架数 3：最小自锁核

Def（骨架判定） 在二维或三维最密核构型族 K(n) 中，若对任意代表构型，所有合法分割线/面均不存在，则 Bone(n)=True。

合法分割线 L：仿射直线，使 K(n) 两侧点集均非空，且对任一开圆盘 B(c,1)，L∩B(c,1)=∅。

Thm 有且仅有 Bone(3)=True；∀n≠3, Bone(n)=False。

证明略述：二维等边三核无合法 L；n=4 存在合法 L（四盘成菱形时取与短对角线平行、位于两切边间隙的 L）。三维三球中心共面退化为二维情形。可作几何穷举验证。

接口对齐：此“自锁核”与任何因子判定无矛盾，仅属结构分类新层。

### 2.5 Convention：5 与 7 为封闭层范例，非骨架

Convention 5 与 7 不称骨架；其意义为：展示骨架 3 向外扩展的封闭层范例。Bone(5)=False，Bone(7)=False。

## 四、看见本源数结构后，得到什么（定理层与规则层）

### 4.1 并置闭锁：偶数被 2 完全承接

Thm ∀k≥2，2k=2×k ⇒ Class(2k)=Even。全体正偶数构成并置承接族。由 2≡1+1 与承接定义导出。

### 4.2 骨架闭锁：3 的倍数被 3 承接

Thm ∀k≥2，3k=3×k ⇒ Carry(3k)=True。3×1=3 为本体，不作承接对象。

### 4.3 最小闭锁周期与 6k±1 走廊

定义闭锁族 F={n: n≡0(mod 2) 或 n≡0(mod 3)}。覆盖类分解的最小公周期为 lcm(2,3)=6。

模类筛三步：S0=Z_{>3}；S1=S0 去除 2 倍数；S2=S1 去除 3 倍数；则 S2={n: n≡1,5 (mod 6)}，为最小不可再细分候选走廊。引入 5 则周期升至 30。

Corollary ∀p>3, p∈ℙ ⇒ p≡1 或 5 (mod 6) ⇔ p=6k±1。

### 4.4 Convention：5 与 7 为走廊早期成员

5 与 7 作为 6k±1 走廊早期成员，承担封闭层范例；骨架身份严格限于 3。

## 五、质数与合数如何判定——承接原则（判定层）

### 5.1 承接的严格定义（结构/接口对齐）

Def Carry(n)=True ⇔ ∃a,b ∈ ℤ, 1<a<n, 1<b<n, n=a×b。
接口对齐：与“存在非平凡因子”在真值上等价。

### 5.2 可承接态与不可承接态

- Carry(n)=True ⇒ n 为合数；
- Carry(n)=False ⇒ n 为质数。

### 5.3 证据范式升级

在证据链层，存在需绑定证据对象 〈因子对，证明算子，复核日志〉，形成可复核证书。真值等价不意味着证据范式等价。

### 5.4 走廊内的承接判定（计算接口）

对 n>3 且 n≡±1(mod 6) 的候选数，仅需在走廊内的质数因子试除至 √n。2 与 3 无需再测。

## 六、新体系的解释力：π 与 e（投影层）

初值选择原则（冻结） 初值须由最小闭锁构型直接给出，并与经典逼近序列首段同构；不符合者（如 π=4+δ′）不采用。

### 6.1 π：闭合投影残值

经典：π = C/D。
新解读：以单位圆内接正 6·2^n 边，初始周径比为 Π0=6/2=3。定义 Πn 为 n 层逼近，π=lim_{n→∞} Πn。

关键表达式：π = 3 + δπ，其中 δπ = lim_{n→∞} (Πn − 3)。

结构解读：3 为唯一骨架数在闭合投影中的初始锁定；δπ 为有限骨架在连续闭环中锁不尽的残差。

### 6.2 e：展开投影残值

经典：e = lim_{n→∞} (1 + 1/n)^n = ∑_{n=0}^∞ 1/n!。

新解读：级数前两项 1+1=2 为并置态展开初值。定义 δe = ∑_{n=2}^∞ 1/n!，则

e = 2 + δe。

结构解读：2 为并置锁定的展开初值；δe 为阶乘展开路径的无穷残差。

### 6.3 π 与 e 的同源关系（要点）

- 本体动作：π 为闭合，e 为展开；
- 初始显影：π 的 3（正六边形周径比），e 的 2（1+1）；
- 结构铰链：6 = 2 × 3（并置与骨架首次耦合）。

## 七、三大问题的同根解释（命题层）

### 7.1 哥德巴赫猜想：双不可承接覆盖问题

经典表述：∀N>2, N≡0(mod 2) ⇒ ∃p,q∈ℙ, N=p+q。

本源重述：偶数 N=2m 为并置态闭合。问题等价为：每个并置闭合态是否均可被两个不可承接态之和覆盖？

N = p+q，Carry(p)=False，Carry(q)=False。

走廊配对结构（由 §4.3 导出）：
- N≡0 (mod 6)：C₁ + C₅；
- N≡2 (mod 6)：C₁ + C₁；
- N≡4 (mod 6)：C₅ + C₅；
其中 C₁={n: n≡1(mod 6)}，C₅={n: n≡5(mod 6)}。

证据对象：
G(N) = { p | p≤N/2, Carry(p)=False 且 Carry(N−p)=False }。
若 G(N)≠∅，则 N 满足猜想；反例为 G(N)=∅。

计算接口：候选 p 仅遍历 6k±1 走廊且 p≤N；验证 N−p 同属走廊并做素性判定至 √(N−p)。2 与 3 无需再测。

结构解释：偶数作为并置闭合态，其结构刚性由两个不可承接态补偿；走廊配对给出三类模 6 方案。

### 7.2 黎曼猜想：不可承接谱平衡问题

经典表述：ζ(s) 非平凡零点均满足 Re(s)=1/2。

经典意义：控制不可承接态密度的全局涨落级别。

本源重述：不可承接态在生成序中的全局涨落是否受“1/2 对称中线”的谱平衡约束？

接口对齐草图：设 χ 为素数指示函数，对其作狄利克雷级数/梅利变换得 F(s)=∑ χ(n)/n^s。临界带 (0,1) 的自对偶对称 s↦1−s 之不动直线为 Re(s)=1/2。双态对称中线在此接口上对应为“对偶对称的不动集”，非“数值巧合”。

结构解释：1/2 在本体系中对应双态对称的中线；若零点全落 1/2，则对应谱平衡被满足；否则对应关系需修正。

### 7.3 哥德尔不完备定理：形式壳不完备边界

经典：一致、可有效公理化且足够表达算术的系统中存在“真但不可证”的命题。

本源定位：不完备是“固定形式壳”的固有限界，而非数学整体的终极缺陷。应以“证据链制度”管理：公理、定义、证明、算法、证书并存，并以状态标注（OriginCarried / ClassicalValid / PendingProof）取代伪装定理。

## 九、证伪条件（可检验性边界）

凡不可证伪的断言，不进入严肃讨论。任一条被严格验证为真，则本文核心断言不成立。

### 9.1 存在大于 3 的第四骨架数

被检验断言：有且仅有 3 为骨架数。

证伪条件：在开单元规则与最密核构型族 K(n)（生成规则 Γ：最大化接触数、最小化外周长度的局部极小代表）下，找到 n>3，使所有代表构型均无合法分割线/面（Bone(n)=True），且不能解释为 3 的倍叠承接态。

算法提示：对 n≤12 可启发式生成 K(n)，对每个 K(n) 执行合法分割判定器。

### 9.2 哥德巴赫反例

被检验断言：每个大于 2 的偶数可表示为两个不可承接态之和。
证伪条件：存在 2N>2，使 G(2N)=∅。

### 9.3 黎曼偏离 1/2 的非平凡零点

被检验断言：非平凡零点实部 1/2 对应双态中线的接口对齐。
证伪条件：存在 s0=β+iγ，0<β<1 且 β≠1/2，满足 ζ(s0)=0。

证据链模板：需提供（i）高精度算子与误差界（如 Odlyzko 型）；（ii）区间零点计数与隔离证明（Riemann–von Mangoldt 一致性）；（iii）独立实现复核报告。仅有数值近似不构成证伪。

### 9.4 旧型不完备性在新地板下等价重现

被检验断言：哥德尔不完备与“孤立一”的地板选择相关。
证伪条件：在采用本源壹与证据链制度的新地板下，仍不可避免地产生与旧哥德尔结构同型的“真但不可证”命题，且无法通过证据链扩展机制化解。
